## 🌟 About GHRAM CENTER

**GHRAM CENTER** is a **full-featured salon & studio management system**, designed to streamline **bookings, staff, services, sales, expenses, and reporting**. It also supports **invoice printing** and quick operations for busy salons and studios.  

This platform is **responsive, fast, and production-ready**, integrating all aspects of day-to-day salon management in one modern frontend solution.

---

## 🚀 Features

### User / Client
- Real-time appointment bookings  
- View available services & staff schedules  
- Multi-language support  
- Notifications for bookings and updates  

### Admin / Salon Management
- Manage **services, staff, and clients**  
- Track **sales, expenses, and revenue**  
- Generate and print **professional invoices**  
- Detailed reports and analytics  
- Quick workflow for busy studios  

### Utilities & Enhancements
- Smooth UI interactions  
- Sliders & carousels with Swiper  
- Form validation and React Select inputs  
- Toast notifications for success/error messages  
- Universal-cookie for state persistence  

---

## 💻 Tech Stack

**Frontend:**  
- React 18  
- Redux Toolkit for state management  
- React Router v6 for navigation  
- TailwindCSS & Styled Components  
- MUI & Ant Design components  

**Utilities / Libraries:**  
- React Toastify for notifications  
- React-to-Print for invoices  
- React Flatpickr for date pickers  
- MUI Datatables for tables & reports  
- Swiper for carousels  
- Web Vitals for performance
